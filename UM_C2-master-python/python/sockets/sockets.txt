TCP handshake (en general)
cliente                         Servidor
   |                               |
   |  -> syn                       |
   |               <- ack + syn    |
   |  -> ack                       |
   |                               |
   |     ... trafico ...           |
   |     ... trafico ...           |
   |     ... trafico ...           |
   |     ... trafico ...           |
   |                               |
   |                               |
   |  -> fin                       |
   |              <- ack + fin     |
   |  -> ack                       |
   |                               |
   |                               |


TCP handshake (server.py - client.py)
cliente                         Servidor
   |                               |
   |  -> syn                       |
   |               <- ack + syn    |
   |  -> ack                       |
   |                               |
   |     ...< Thank you...<        |
   |                               |
   |              <- ack + fin     |
   |  -> ack                       |
   |  -> fin + ack                 |
   |              <- ack           |
   |  -> ack                       |
   |                               |
